🫐 Wild Blueberry Yield Prediction

This project predicts wild blueberry yield using machine learning.
It analyzes environmental and bee-activity factors like temperature, rainfall, and fruit mass to estimate total yield.

⚙️ Tools Used

Python · Pandas · NumPy · Scikit-learn · Matplotlib · Seaborn
